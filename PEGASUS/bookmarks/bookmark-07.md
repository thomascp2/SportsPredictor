## run_daily.py — Architecture Notes

### Entry point
```bash
python PEGASUS/run_daily.py                    # today, smart-only
python PEGASUS/run_daily.py --date 2026-04-15  # specific date
python PEGASUS/run_daily.py --all-picks        # skip is_smart_pick filter
python PEGASUS/run_daily.py --sport nba        # single sport
python PEGASUS/run_daily.py --dry-run          # no JSON write
```

### Flow
```
1. _check_readiness() → queries predictions + smart counts per sport
   - predictions_ready: total rows >= 10
   - pp_sync_ready:     is_smart_pick=1 rows >= 1
   - Exits 1 if predictions missing AND smart_picks_only=True

2. get_picks(smart_picks_only=True) → List[PEGASUSPick]

3. _write_json() → PEGASUS/data/picks/picks_{date}.json
   - Contains: pegasus_version, game_date, meta{readiness, tier_counts, ml_blended}, picks[]
   - Never writes to SQLite, Supabase, or Turso

4. _print_summary() → terminal output
```
