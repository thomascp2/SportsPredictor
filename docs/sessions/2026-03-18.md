## Key Decisions

### NHL blowout filter uses moneyline, not spread
Puck line is always ±1.5 — no mismatch signal. Moneyline implied probability > 78%
(~-355 moneyline) is the blowout threshold. Tonight's games ranged 55–59% — nothing filtered.

### NHL odds via ESPN summary endpoint
NHL scoreboard API doesn't return odds. Had to fetch per-game summary (pickcenter array).
One API call per game — fine for 6–10 games per night. 6/6 matched tonight.

### Canonical key compatibility fix
Branch's statistical_predictions_v2.py writes f_lambda_param, f_l10_avg, f_std_dev.
Master's smart_pick_selector read lambda_param, sog_l10, std_dev.
Fixed: selector now tries canonical keys as fallback for forward compat.
