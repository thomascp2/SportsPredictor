**OAuth Flow Fix**
- **Root issue:** `supabase.auth.signInWithOAuth()` alone doesn't work in React Native ‚Äî no browser redirect handling
- **Fix:** Rewrote `authStore.ts` to use `expo-web-browser` + `expo-auth-session`:
  1. Call `signInWithOAuth` with `skipBrowserRedirect: true` ‚Üí get OAuth URL
  2. Open URL via `WebBrowser.openAuthSessionAsync(url, redirectTo)`
  3. On callback: `QueryParams.getQueryParams(url)` ‚Üí `supabase.auth.setSession()`
  4. `makeRedirectUri()` generates `exp://192.168.x.x:8081` (Expo Go) or `freepicks://` (standalone)
