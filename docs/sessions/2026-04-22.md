import duckdb, sqlite3, unicodedata
from datetime import date, timedelta

def norm(n):
    if not n: return ''
    x = unicodedata.normalize('NFKD', str(n))
    return ''.join(c for c in x if not unicodedata.combining(c)).lower().strip()

## New Player / Callup Policy (documented in `mlb_feature_store/CLAUDE.md`)
- New players are **excluded from ML predictions, not imputed**. Dashboard shows `—` for ML columns. Stat model still covers them.
- Self-healing: Statcast ingests their first game ~24h after it's played. `run_daily.py` picks it up. Next `predict_to_db` run includes them.
- `name_aliases` table auto-populates on match. If a new player shows `NO FEATURES` for more than 3 days after their debut, check if their Statcast name matches the stat model name.
- **Do NOT** add league/team average imputation for debut players. The model accuracy stats are only valid for players with real rolling Statcast features.
